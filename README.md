# نگین جواهر — Negin Jewelry

پورتفولیوی طراحی برای یک برند خیالی جواهرات دست‌ساز، ساخته‌شده با Next.js (App Router + TypeScript)، Tailwind CSS و Framer Motion.

## اجرا

```bash
npm install
npm run dev
```

سپس آدرس `http://localhost:3000` را باز کنید.

## کانسپت طراحی

**شخصیت برند:** آتلیه — نه ویترین شرکتی. جای کلیشه‌ی همیشگی طلایی‑مشکی، این طراحی حول ایده‌ی «دست، قبل از ماشین» شکل گرفته: زمینه‌ی تیره‌ی گرم (نه مشکی خالص)، لهجه‌ی برنزی/طلایی محدود و هدفمند، و یک قرمز یاقوتی عمیق برای لحظات کلیدی.

**پالت رنگ:**
- `#15120F` ink — زمینه‌ی اصلی، اسپرسوی گرم
- `#EFE7D8` porcelain — متن اصلی، عاجی گرم
- `#B08A46` brass — لهجه‌ی برنزی برای CTA و اعداد
- `#6E1F2A` garnet — لهجه‌ی یاقوتی برای لحظات خاص

**تایپوگرافی:** فونت Vazirmatn (متغیر) برای عنوان و متن — خوانایی بالا برای فارسی، بدون هیچ letter-spacing روی حروف فارسی.

**امضای صفحه:** نماد «نگین برش‌خورده» (Faceted Gem) — یک الماس هندسی که با Framer Motion خط‌به‌خط ترسیم می‌شود، دقیقاً مثل نوری که روی تراش یک نگین واقعی می‌افتد. این نماد به‌عنوان لوگو، جداکننده‌ی بخش‌ها و پس‌زمینه‌ی تزئینی در سراسر سایت تکرار می‌شود و مستقیماً به معنای «نگین» در نام برند برمی‌گردد.

## ساختار پروژه

```
app/            صفحه اصلی و layout
components/     کامپوننت‌های هر بخش (Header, Hero, Collection, ...)
lib/data.ts     محتوای فارسی و اطلاعات محصولات
```

## نکات فنی

- تمام تصاویر از Unsplash و با مجوز استفاده تجاری رایگان هستند.
- طراحی به‌طور کامل RTL و ریسپانسیو (موبایل، تبلت، دسکتاپ) است.
- فرم مشاوره فعلاً فقط رابط کاربری را نمایش می‌دهد (بدون اتصال به بک‌اند)؛ برای اتصال واقعی باید یک API route یا سرویس فرم مثل Formspree اضافه شود.
